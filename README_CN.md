## 👁️ 重新定义隐私

在这个时代，把绝密数据交给任何云盘（iCloud、Google Drive、各种企业网盘）都意味着将刀柄递给他人。**我们不信任任何云，我们只信任数学。**

Phantom-FS 并不是一个提供存储空间的网盘。它是一个存在于设备内存中的**赛博粉碎与重组引擎**。
它将你的机密文件（财务报表、绝密视频、私钥账本）切成无数个高熵值的"加密碎沙"，散落在网络上。没有你的允许，这些数据在任何人眼里都只是毫无意义的电子白噪声。

## ⚔️ 核心特性

### 1. 云端降维：让所有网盘沦为"蠢硬盘"
无论你是把加密碎片存在本地、私有云还是公有云，服务器都无法知晓你存了什么、文件名是什么。即便云服务商被黑客彻底脱库，泄露的也只是一堆无法解密的废料。

### 2. 物理级冷钱包：二维码藏宝图
最高级别的秘密，不应该存在 U 盘里。加密完成后，Phantom-FS 会生成一张几 KB 的**高密度二维码（Manifest 图纸）**。
你可以将它打印在纸上，锁进银行保险箱。解密时，只需用任意浏览器扫码。真正的数字资产物理隔离，防断电、防黑客、防硬件老化。

### 3. O(1) 极速播放：大文件的毫秒级穿梭
得益于革命性的**「确定性 IV 推导算法」**，当你解密一部 50GB 的加密机密视频时，无需漫长的等待。在进度条上随意拖拽，前端引擎能在毫秒内瞬间推导出局部密文并解密播放。**体验如丝般顺滑，全程内存闭环。**

### 4. 阅后即焚：内存里的幽灵
当你关闭浏览器的瞬间，系统会发出硬件级指令，将内存中的解密画面彻底覆写清零。拔网线即焚，不留一丝残影。

## 🔐 三重物理边界

要复原一份数据，必须同时集齐以下三个物理维度的要素，缺一不可：

1.  **箱子（Chunks）**：散落在云盘或硬盘里的二进制加密碎片。
2.  **藏宝图（.ptm 图纸 / 二维码）**：记载了碎片的顺序与动态盐值。
3.  **唯一钥匙（你的脑中密码）**：绝不接触网络，仅在使用瞬间于本地内存中派生出 256 位主密钥。

> *"就算有人拿着枪逼着交出服务器，我们能交出的也只是一堆打不开的沙子。因为解密的钥匙，从来只在你的脑子里。"*
